# VSCode-settings

## Installing
1) Install VSCode (if you are not installed VSCode before)
2) Clone this repo
3) Installing plugins by command: python vsinstall.py

> git clone https://github.com/32-52/VSCode-settings.git
>
> cd VSCode-settings
>
> python vsinstall.py
